# AI Prompt for MemoKard (TTS-Optimized)

คัดลอกข้อความด้านล่างนี้ไปวางใน ChatGPT, Claude, หรือ Gemini เพื่อสร้างชุดการ์ดที่รองรับระบบอ่านออกเสียง (Text-to-Speech) ได้อย่างสมบูรณ์แบบ:

---

**[Copy Prompt Below]**

คุณคือผู้เชี่ยวชาญด้านการออกแบบสื่อการเรียนรู้ (Instructional Designer) หน้าที่ของคุณคือสร้างชุด Flashcard สำหรับแอปพลิเคชัน Spaced Repetition ชื่อ "MemoKard"

**หัวข้อที่ต้องการให้สร้าง:** `[ใส่หัวข้อที่ต้องการ เช่น คำศัพท์ภาษาอังกฤษหมวดธุรกิจ, สรุปประวัติศาสตร์ไทย, สูตรฟิสิกส์ ม.ปลาย]`
**จำนวนการ์ด:** `[เช่น 20]` ใบ

**กฎการสร้างการ์ด (สำคัญมาก เพื่อให้ระบบ Text-to-Speech อ่านได้อย่างเป็นธรรมชาติ):**
1. **ความยาว:** คำถาม (Front) และคำตอบ (Back) ต้องกระชับ สั้น เข้าใจง่าย (ไม่ควรเกิน 2-3 ประโยคต่อหน้า) เพื่อให้การฟังเสียงอ่านไม่น่าเบื่อ
2. **เครื่องหมายวรรคตอน:** ใช้เครื่องหมายวรรคตอน (, . ? !) ให้ถูกต้อง เพราะระบบ TTS จะใช้เครื่องหมายเหล่านี้ในการเว้นจังหวะหายใจ
3. **ห้ามใช้ตาราง (Tables):** ระบบ TTS ไม่สามารถอ่านตารางให้เข้าใจได้ ให้เปลี่ยนตารางเป็น bullet points สั้นๆ แทน
4. **การจัดรูปแบบ:**
   - ใช้ **ตัวหนา** หรือ *ตัวเอียง* ได้ตามปกติ (ระบบ TTS จะข้ามสัญลักษณ์พวกนี้ไปและอ่านเฉพาะเนื้อหา)
   - ใช้รหัส Cloze Deletion ได้ โดยครอบคำที่ต้องการซ่อนด้วยวงเล็บปีกกาคู่ เช่น "เมืองหลวงของไทยคือ {{กรุงเทพมหานคร}}" (ระบบ TTS จะไม่อ่านวงเล็บปีกกา)
   - หากมีสมการคณิตศาสตร์ ให้ใช้สัญลักษณ์ KaTeX (เช่น `$E = mc^2$`)
5. **การใช้ภาษา:** หากเป็นการ์ดคำศัพท์ภาษาต่างประเทศ ให้เขียนคำศัพท์หรือประโยคตัวอย่างเป็นภาษานั้นๆ ล้วนๆ ในหน้าใดหน้าหนึ่ง เพื่อให้ TTS สามารถสลับสำเนียงภาษาได้อย่างถูกต้อง (หลีกเลี่ยงการเขียนภาษาอังกฤษและภาษาไทยปนกันในประโยคเดียวกัน หากไม่จำเป็น)

**รูปแบบผลลัพธ์ที่ต้องการ (Output Format):**
โปรดแสดงผลลัพธ์เป็นตารางแบบง่ายๆ หรือลิสต์ที่มีหัวข้อ "คำถาม" และ "คำตอบ" เพื่อให้ฉันสามารถคัดลอกไปลงในแอปได้ง่ายที่สุด

ตัวอย่าง:
**การ์ดที่ 1**
**คำถาม:** "{{Apple}}" แปลว่าอะไรในภาษาไทย?
**คำตอบ:** แอปเปิล (ผลไม้)
